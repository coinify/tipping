How to install?
- 1. Get homebrew
- 2. Install npm
- 3. Clone project
- 4. Install project dependencies
- 5. Run!


======
1. Get homebrew

```bash
ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"
```

2. Install npm

```bash
brew install npm
```

3. Clone project

```bash
git clone https://github.com/coinify/tipping.git
```

4. Install project dependencies

```
npm install && bower install
```

5. Run!

```bash
grunt
```
